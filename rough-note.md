Saveswitch is a web application that allows users to save copied resources from one device and use them on another device, with no mandatory
login or sign up process. thanks to the (Xoomshare) Feature. 
WHAT IS XOOMSHARE? 
Xoomshare is a feature that allows users to save copied resources from one device and use them on another device, with no mandatory login or
sign up process. 
how does it work? 
1. User copies a text, link or image on a device 
2. User opens the Saveswitch web app on the device 
3. User clicks the “Xoomshare” button on the login page or sign up page 
4. User is redirect to a new page to create a unique path code e.g (saveswitch.xyz/my-secret-room) for the
copied text, link or image Xoomshare page 
5. The web app saves the copied text, link or image to the cloud 
6. The receiver device retrieves the copied text, link or image from the cloud through the unique path code e.g
(saveswitch.xyz/my-secret-room) 
7. The user can now use the copied text, link or image on the receiver device 
User flow 
1. User copies a text or link on a device 
2. User opens the Saveswitch web app on the device 
3. User clicks to login using google auth or simply use the “Xoomshare” feature without login 
4. The web app saves the copied text or link to the cloud 
5. The receiver device retrieves the copied text, link or image from the cloud through the unique path code e.g
(saveswitch.xyz/my-secret-room) 
6. The user can now use the copied text, link or image on the receiver device without the hassle 
Signed up user flow 
1. User copies a text or link on a device 
2. User opens the Saveswitch web app on the device 
3. User clicks to login using google auth 
4. The web app saves the copied text or link to the cloud 
5. User can now access the copied text or link on the receiver device when logged in 
What is the Private and Public space switch? 
at the top middle of the web app there is a switch that allows users to switch between private and public space. the public space is for sharing
text, links or images with the world when their user name path is been clicked e.g saveswitch.xyz/@username where people can see your
saved resources while the private space is for saving text, links or images for only you to see. as a user you can switch between private and public
space at any time and at any place. when you switch to public space, your resources will be visible to the public when they click on your username
path and when you switch to private space, your resources will be visible to you alone. 
What is the comment space? 
at the right bottom of the web app there will be a comment space that allows users to comment on your public shared resources. but this feature
is only available for signed up users. when a user comments on your saved resources, you will receive a notification on your dashboard
